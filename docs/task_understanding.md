# Task Understanding

## Background

无人机视觉感知系统需要在复杂空中视角下完成目标检测、跟踪和智能分析。核心挑战包括小目标、尺度变化、实时性、光照天气干扰、运动模糊和目标类别差异。

## Planned Stages

1. 环境准备与数据理解：熟悉 Python、PyTorch、OpenCV，完成 VisDrone 和 UAV123 数据探索。
2. 目标检测训练与评估：训练 YOLO、Faster R-CNN 或 SSD，并评估 mAP、FPS、小目标与遮挡场景表现。
3. 目标跟踪算法实现：实验 SORT、DeepSORT 或 Siamese 方法，在 UAV123 上评估成功率和精确度。
4. 模型优化与集成：尝试轻量化、剪枝、量化，将检测和跟踪模块整合为完整流程。
5. 总结与展示：整理代码、实验记录、报告和演示素材。
